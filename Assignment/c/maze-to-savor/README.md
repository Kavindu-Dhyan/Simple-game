# Maze to Savor

## Overview
"Maze to Savor" is an engaging maze exploration game where players navigate through intricately designed mazes, solve puzzles, and overcome challenges. The game emphasizes strategy, exploration, and player interaction.

## Features
- **Maze Generation**: Dynamic maze creation with various difficulty levels.
- **Player Movement**: Intuitive controls for navigating the maze.
- **Game Logic**: A robust game loop that manages the game state and player interactions.
- **Utility Functions**: Helper functions for random number generation and input handling.

## Getting Started

### Prerequisites
- A C compiler (e.g., GCC)
- Make utility

### Building the Project
To compile the game, navigate to the project directory and run:
```
make
```

### Running the Game
After building, you can run the game with:
```
./maze-to-savor
```

## Directory Structure
- `src/`: Contains all source code files.
  - `main.c`: Entry point of the game.
  - `game/`: Contains game logic implementation.
  - `maze/`: Contains maze generation and manipulation functions.
  - `player/`: Contains player-related functions.
  - `utils/`: Contains utility functions.
  - `features/`: Documentation for additional features.
- `include/`: Centralized header files for easy inclusion.
- `Makefile`: Build instructions for the project.
- `README.md`: Project documentation.
